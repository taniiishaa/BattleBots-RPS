An interactive, terminal-based Rock Paper Scissors game that pits human intuition against a randomized AI. This project demonstrates core programming concepts like **conditional logic**, **input validation**, and **persistent loops**.

## üöÄ Features

* **Smart AI Opponent:** Uses the `random` module to ensure unpredictable gameplay.
* **Robust Input Validation:** Prevents program crashes by handling invalid user inputs gracefully.
* **Human-Readable Interface:** Maps numeric data (1, 2, 3) to clear descriptive text for a better user experience.
* **Session Persistence:** Includes a "Play Again" loop to allow for continuous gameplay sessions.
